### Application

# install libraries

```
npm install
```

# start application server

```
npm start
```
> http://localhost:3000](http://localhost:3000)


### DataBase

# install libraries

```
cd npm
npm install
```

# start db server

```
nodemon server
```
> http://localhost:4000](http://localhost:4000)
