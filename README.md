# Manager Application

### library installation

```
npm install
```

### start application server

```
npm start
```
> [http://localhost:3000](http://localhost:3000)


# Manager DataBase

### library installation

```
cd npm
npm install
```

### start db server

```
npm install -g nodemon
nodemon server
```
> [http://localhost:4000](http://localhost:4000)

### `mongodb://localhost:27017/manager`
